# Database

Thư mục này dành cho tài liệu và script SQL Server.

## Nên chứa gì ở đây
- `init.sql`: tạo database, bảng, quan hệ
- script thêm dữ liệu mẫu
- tài liệu ERD hoặc mapping bảng

## Quy ước gợi ý
- `01_create_database.sql`
- `02_create_tables.sql`
- `03_seed_data.sql`

Nếu đồ án nhỏ, có thể giữ một file `init.sql` duy nhất.
